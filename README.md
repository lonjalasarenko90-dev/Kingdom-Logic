# Kingdom Logic

Kingdom Logic is an offline-first web game designed to train clarity of thought, iron logic, integrity, and conscious decision-making.

Built as a single HTML artifact, it runs fully offline and can be deployed instantly via GitHub Pages.

---

## 🎯 Features

- 🧠 30 structured missions (Logic, Integrity, Life)
- 🔎 No guessing — only reasoning and consequences
- 📈 Real-time story progress bar
- 📅 Daily Mode with randomized missions
- 🔥 Streak system
- ⎘ Share button (copy results instantly)
- 🌐 Fully offline (no external dependencies)
